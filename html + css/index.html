Aggie Path

Welcome to Aggie Path, a UC Davis AI Schedule generator!

What is Aggie Path?

Aggie Path is an AI class recommendation system tailor made to help UC Davis Computer Science students find their optimal schedule. This project was made by students for students.

The system uses a few different mechanisms. The system uses contrait-based optimization and rule-based filtering to address hard and soft constraints. The system employs XGBoost for decision making. Lastly, the system utilizes a decision tree to validate the system with example transcripts.

How to use Aggie Path:

  1. Upload a PDF of your most recent transcript
  2. Aggie Path will consider your degree requirements and schedule constraints
  3. Aggie Path will give you your recommended schedule

it's that easy!

Aggie Path going forward

If this project is able to go further, it would be expanded to include students from all UC Davis majors. The application would also consider adding the following:

  1. adding a minor or second major to your degree requirements
  2. further personalization
  3. schedule generation further than a quarter
